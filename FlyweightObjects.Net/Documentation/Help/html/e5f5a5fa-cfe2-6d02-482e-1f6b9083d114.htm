ServiceContextBase..::TrySelect<(Of <(T>)>) Method (IStorageCommand, T%)
FlyweightObjects.NET
Returns a boolean value indicating whether or not the source exists in storage. If true, the source will be the retrieved object, otherwise it will be its default. Note the value for sql may be any object which implements the IStorageCommand interface including StoredProcedure.
