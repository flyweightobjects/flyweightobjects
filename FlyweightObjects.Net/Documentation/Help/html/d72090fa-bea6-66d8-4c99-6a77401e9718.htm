CacheManager..::AddObject<(Of <(T>)>) Method
FlyweightObjects.NET
Adds a new object to the cache. Be sure to call the Contains method to circumvent any key conflicts. This method will hold objects in memory which will self-expire provided that their time-to-live value is greater than zero.
